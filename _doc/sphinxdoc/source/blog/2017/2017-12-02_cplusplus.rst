    :title: C++ depuis un notebook
    :keywords:
    :date: 2017-12-02
    :categories: langage

    Il est maintenant possible de faire du C++ dans un notebook :
    `Interactive Workflows for C++ with Jupyter <https://blog.jupyter.org/interactive-workflows-for-c-with-jupyter-fe9b54227d92>`_.
    Ce langage s'ajoute à l'incroyable liste déjà disponible :
    `Jupyter kernels <https://github.com/jupyter/jupyter/wiki/Jupyter-kernels>`_.
    Ce petit tour de force repose sur
    `Cling <https://root.cern.ch/cling>`_,
    un interpréteur pour C++ et
    `Xeus <https://github.com/QuantStack/xeus>`_
    qui implémente le kernel.
    L'article mentionne d'autres outils très sympathiques.
    Parmi eux, `xtensor <https://github.com/QuantStack/xtensor/>`_
    qui permet le calcul numérique sous C++.
    `mybinder <https://mybinder.org/>`_ permet
    de `dockeriser <https://fr.wikipedia.org/wiki/Docker_(logiciel)>`_
    un repository github.
