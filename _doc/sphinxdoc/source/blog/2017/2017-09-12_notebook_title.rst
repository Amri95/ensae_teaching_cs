    Le site contient plus de 300 notebooks qu'on peut
    retrouver compilés sous la forme d'une gallerie
    :ref:`l-notebooks` et d'une page qui indique la couverture
    des tests unitaires : :ref:`l-notebooks-coverage`.
    Excepté si les notebooks utilisent un cluster un une autre
    ressources difficile d'accès,
    ceux-ci sont exécutées une fois par semaine pour vérifier
    qu'ils continuent à tourner avec les dernières versions
    des modules dont ils dépendent.
    Ce faisant, le titre est maintenant précédé d'un code qui indique
    le public auquel il s'adresse et le contenu.
    Il y a trois niveaux, 1A, 2A, 3A pour chaque année du cursus
