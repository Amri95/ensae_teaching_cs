    Les notebooks de deep learning devenaient difficiles
    à maintenir dans le même projet, les librairies
    dépassent souvent la centaine de mégaoctets
    et allongeaient considérablement le temps d'exécution
    des tests unitaires. Ils ont désormais leur
    propre projet :epkg:`ensae_teaching_dl`.
