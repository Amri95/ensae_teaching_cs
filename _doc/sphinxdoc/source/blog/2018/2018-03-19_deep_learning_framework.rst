    C'est un article de blog produit par la recherche de Microsoft :
    `Comparing Deep Learning Frameworks: A Rosetta Stone Approach <https://blogs.technet.microsoft.com/machinelearning/2018/03/14/comparing-deep-learning-frameworks-a-rosetta-stone-approach/>`_.
    Il compare les temps d'apprentissage et de test pour la plupart
    des librairies de deep learning.
    L'ensemble est disponible sur
    `GitHub <https://github.com/ilkarman/DeepLearningFrameworks>`_.
    Le répertoire de
    `notebooks <https://github.com/ilkarman/DeepLearningFrameworks/tree/master/notebooks>`_
    est intéressant car cela permet de récupérer les mêmes exemples (4)
    implémentés avec les différantes librairies de deep learning
    les plus connues.
